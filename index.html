Marble Mania

Built using Unity, Blender3D and Substance, Marble Mania is a tribute to the 1984 classic "Marble Madness" and was made to further my learning in Unity, specifically around the use of visual effects, mesh manipulation in real time and enemy AI.
Also it was really fun to make.

What Happened in Tawley

W.H.I.T is a first person narrative-driven survival game built in Unity.
This is a long-form and is very scenery-heavy, so it takes a long time to progress.
The purpose of this game is to learn more about physical interactions, atmosphere and suspense.
