Game Projects ⬆

Marble Mania

Built using Unity, Blender3D and Substance, Marble Mania is a tribute to the 1984 classic "Marble Madness" and was made to further my learning in Unity, specifically around the use of visual effects, mesh manipulation in real time and enemy AI.
Also it was really fun to make.

ConcentryX

ConcentryX is a hyper-casual rotary break-out style score-attack shooter game built in Unity.
The purpose of this game is to learn more about automated level creation, prefabs, post-processing effects, performance balancing, garbage collection and enemy creation.

Awkward Robot

AW was the first game I built specifically for mobile devices.
This was made in order for me to learn more about mobile development and play-store deployment, as well as the use of non-standard control schemes such as tilt controls.

W.H.I.T

What Happened in Tawley is a first person narrative-driven survival game built in Unity.
This is a long-form and is very scenery-heavy, so it takes a long time to progress.
